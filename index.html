Projects

Online Ticketing System

|

The Online Ticketing System is a full-stack web application developed using Java Servlet. It allows customers to order tickets simultaneously, and the application uses multi-threading and networking techniques for asynchronous communication between microservices such as customers, orders, event hosts, and email notifications. The system also uses JDBC to establish a connection to an SQLite database to manage and store data.

Social Media Web Application

|

The Social Media Web Application is a complex web application that required a robust back-end system developed using Golang and gRPC. The back-end system incorporates Hashicorp raft for distributed and reliable data storage, ensuring the data is stored safely and redundantly. The project also includes comprehensive unit testing to guarantee the stability and performance of the system.

Easy Campus Life

|

Easy Campus Life is a full-stack integrated campus platform that facilitates student discussions, course group formation, socializing, and event participation. The platform is built using various AWS services such as S3, API Gateway, Lambda, SES, Cognito, DynamoDB, and OpenSearch. The project also includes RESTful APIs for seamless data access and manipulation between the web and the server. The platform uses a NoSQL database management system (DBMS) to efficiently handle large and complex data sets and ensure scalability.

NBA Statistics

|

The NBA Statistics project is a web application developed using Streamlit and written in Python. The application interacts with a DBMS using PostgreSQL, which contains a custom-designed relational database schema with 20+ entities/relations. The application allows users to access and analyze NBA statistics, providing valuable insights into player performance, team performance, and game analysis.

Taiwan Sign Language Recognition System

|

This system translate 40 common Taiwan sign languages into text using regular camera. In this project, I build up the first Taiwan Sign Language dataset for model training. The result reaches 98.64% accuracy through the method of model ensemble.

Action Recognition using Convolution Neural Network

|

I build a model that combines Yolov4 as the object detection model and I3D as the action recognition model and adopt Actor-Conditioned Attention Maps (ACAM) to take consider of the whole scene. The experiment results on AVA-Kinect dataset demonstrate the effectiveness of our work, results in 24.15 mAP.

Ensemble Method for Twitter GIF Recommendation

|

To recommend GIF categories for tweet responses, our best solution is to design a LSTM model and ensemble with three state-of-the-art pre-trained models: BERT, XLNet and RoBERTa. We achieved 53.45% accuracy on the public test dataset and finally ranked 7/31 in the class.
